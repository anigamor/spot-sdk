The purpose of this branch of the Spot SDK is to brainstorm and develop ideas around using the Spot as an Environmental Consulting Tool.
Current applications include drilling support, geology survey, construction inspection, compliance inspection, and direct sampling. 
Data input is to be Google Earth KML/KMZ, ERSI-type GIS Shapefiles, and CAD Data.
Data is to be output as annotated photos JPG or PNG, ERSI-type GIS Shapefiles, or other data required for the application.
